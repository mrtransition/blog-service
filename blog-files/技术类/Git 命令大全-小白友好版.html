🚀 小白常见场景 · 三步走

📌 场景1: 第一次提交代码
git init → git add . → git commit -m "first commit"
🌿 场景2: 创建新分支并切换
git branch feature → git checkout feature 或 git switch -c feature
☁️ 场景3: 推送到远程仓库
git remote add origin 地址 → git push -u origin main

✨ 小贴士: 随时用 git status 查看状态，用 git log --oneline 简洁查看历史。
