# 轻舟博客（Go 便携版）

一个使用 **Go** 编写的极简本地博客系统，无需数据库，支持 Markdown/HTML 文章，开箱即用。

## 特性
- 纯 Go 实现，编译后仅一个可执行文件
- 自动读取目录结构生成导航
- 支持 Markdown 和 HTML 两种文章格式
- 右侧悬浮文章目录，点击跳转
- 代码高亮、图片相对路径自动适配
- 配置文件驱动，博客名称可自定义
- 首次运行自动生成示例配置和文章

## 快速开始

### 1. 下载可执行文件
从 [Releases]() 页面下载对应系统的 `blog-server` 文件（或自行编译）。

### 2. 运行
```bash
./blog-server -port 3000 -dir blog-files
